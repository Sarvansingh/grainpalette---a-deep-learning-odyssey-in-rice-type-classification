About the Project

This project is inspired by the need for better transparency in the agriculture and food supply chain. Whether you're a farmer, wholesaler, or retailer, quickly verifying rice types is now easier than ever.

We use the MobileNetV4 model, fine-tuned on high-quality labeled images from Kaggle, capable of classifying 5 distinct rice types with over 97% accuracy.
